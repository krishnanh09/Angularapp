Steps to start the application

1. Run the command npm install in CMD
2. Goto the Angular app folder and navigate to data folder and open CMD
3. Try step 4 but if it does not work, run the command --> npm i json-server and then run step 4.
4. Run the command -->  json-server --watch db.json
5. Run the application using npm start
6. Goto localhost:4200 in browser


Note:- To view the application in mobile, change to mobile view using Developer tools and reload from home page
